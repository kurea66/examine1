カラム作成の際カラム名 VARCHAR(255) NOT NULL,nullが入れられなくなる

SELECT カラム名 AS 表示したいタイトル,カラム名 AS 表示したいカラム名 FROM テーブル名;…テーブル内の指定したカラムをASの後ろのカラム名で表示するようにできる。title等を日本語で表示可能

SELECT DISTINCT カラム名 FROM テーブル名…カラム内のものを重複表示しない一覧で表示できる。同じカラム名は１つになる。

CASE カラム名 WHEN THEN END

switch文的な分岐

CASE カラム名 WHEN '比較対象1' THEN '内容'
WHEN '比較対象2' THEN '内容'
ELSE '内容'
END AS カラムの表示名(入れないとCASE文の内容そのまま全部表示される。)
FROM テーブル名

			SELECT title AS 書名,category AS 分類,
			CASE category WHEN '雑誌' THEN '1F'
			WHEN 'コミック' THEN '1F'
			ELSE '2F'
			END AS 階
			FROM books;

CASE WHEN THEN END
