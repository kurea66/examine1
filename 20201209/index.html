コレクションとは

List

順序をつけて並べて格納する。

ArrayList,LinkedList等

Set

順番の概念がなく中身の重複不可！！同じものを登録したくないときに使う

HashSet,LinkedHashSet,TreeSet等

Map

ペアで対応付けて格納する。(ex:京都⇒八つ橋、熊本⇒スイカ等)

HashMap,LinkedHashMap,TreeMap等

上の３つをコレクションフレームワークという

<>記号を使って、格納するインスタンスの型を指定する

ジェネリクスと呼ぶ

インスタンスでないもの(基本型のデータ)は格納できない

基本型のデータを入れる場合はラッパークラスで宣言すること

Javaにはオートボクシング(AutoBoxing)とオートアンボクシング(AutoUnboxing)機能が備わっていて自動変換される。

型名(IntegerやString).valueOf(データ);データを指定された型に変換

型名.parse変換後の型(データ)…parseの後の型に変更

setメソッドは戻り値に上書きされて消えた値を返す。

remove()メソッドは戻り値に削除された値を返す

<>中身が省略されたものをダイヤモンド演算子と呼ぶ
