フォーム

ユーザーに入力を求める場合はformタグで括る！

input type="text"ユーザーにテキストを入力させるための指定。属性text

name="名前"nameの指定は任意　プログラムで認識させるための名前を付ける。　username等

value属性そのフォーム部品を管理する値。

placeholder属性ユーザーが入力する前の初期値。入力したら消えるもの。
例：お名前を入力してください　等

input type="submit" value="送信"input type="submit"で送信ボタンを実装、valueの中はボタンに表示したい文字

input type="number"数値をクリックで移動できるウィンドウを表示、max,min等も設定可能

input type="radio"一個しか選択できないボタン

input type="checkbox"複数選択できるチェックボックスを追加

textarea rows="数値" cols="数値"テキスト入力エリアを作る。　rowsで行　colsで列

関数、タグ

type="date"日付選択。右側にクリックでカレンダー押せる！！

"datetime-local"ローカル日時　カレンダーも押せるよ！

Math.abs(値)絶対値を返す
